BasketScreenActions

sealed class BasketScreenActions

Represents the actions that can be performed on the basket screen.

This sealed class defines various actions that can be triggered on the basket screen, including loading basket items, calculating basket totals, updating basket item count, toggling recommended products, making recommendations, and checking out items.
